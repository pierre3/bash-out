# ブラウザゲーム仕様

## 概要
ブラウザで動作する2Dボスバトル型ブロック崩しゲーム。HTML5 Canvas / TypeScriptで実装する。
従来のブロック崩しをベースに、敵キャラクターとの攻防・エネルギーシステム・キャラクター固有の必殺技による戦略的な駆け引きが特徴。

## 技術方針
- ブロック崩し実装に特化した最小限の抽象化から始め、必要に応じて汎用化する
- 描画: Canvas 2D をベースとし、エフェクト表現のため WebGL 導入も視野に入れる
- 物理演算: 精緻な計算より、計算負荷・実装しやすさ・保守性を重視した最適な方法を選択
- 画面解像度: 内部論理解像度は 540×960（9:16）に固定。CSSでウィンドウサイズに合わせて等比スケール（レターボックス）。devicePixelRatio 対応で高DPI端末でもクリスプに描画

## ゲームエンジン
- ゲームループ（Update / Draw）
- スプライト管理、衝突判定、入力処理
- シーン管理、ゲームオブジェクトのライフサイクル
- TypeScriptによるクラス構造
- ゲームオブジェクト: 位置・速度・スプライトなどのプロパティ、Update/Drawメソッド

## ゲームコンセプト

画面上部に陣取る敵キャラクターとの1対1のボスバトル。
プレイヤーはパドルでボールを跳ね返してブロックを破壊し、敵にダメージを与える。
敵は時間経過でブロックを追加し、様々な攻撃でプレイヤーを妨害してくる。
ブロックから落下する★を集めると、選択キャラクター固有の必殺技が発動する。

## ゲームプレイ

### 勝敗条件
- **勝利**: 敵のHPをゼロにする
- **敗北**: 以下いずれか
  - ライフ（3機）を全て失う（ボールを画面下に落とす）
  - ブロックがパドルの少し上のラインまで到達する

### 攻撃手段（プレイヤー → ボス）
ブロックを壊すこと自体ではボスにダメージは入らない。ブロックは**ボスを守る盾**として機能する。
- **ボールがボスに直撃**: 小ダメージ（**3**）、単純反射で跳ね返る
- **必殺技弾がボスに直撃**: 大ダメージ（**25**、HPの約4割）。1発の発射で1回まで
- ★アイテム収集→必殺技は「ブロック列の貫通」と「ボス直撃」を1発でこなせる
- 通常時はブロックの隙間からボールをボスに通すのが基本戦術

### 敵キャラクター
- 画面右上に配置され、HPバーが画面上端に横バー表示される
- HP=60。ボール/必殺技弾の直撃のみで減る
- 4種類の攻撃パターン（全ての攻撃に1〜2秒の予兆）
  - **電撃**: パドルを短時間（約1秒）停止させる
  - **ボール加速**: 飛んでいるボールを一気に加速（蓄積。ミスでリセット）
  - **ブロック強化**: 既存ブロックからランダムに1〜2個選びHP+1（岩化）
  - **ボディアタック**: ジャンプ→落下で着地時に最上段にブロック追加。**着地中はボスがプレイ領域に降りてくるため判定が下がり、ボールヒットの隙が生まれる**
- 攻撃間隔はHPが減るほど短くなる（idle時間 5秒 → 2秒で線形補間）

### ボールとパドル
- パドルは曲面反射（当たり位置で角度変化）
- ボールの速度はゲーム進行とともに上昇
- ボール落下時はパドル上に即座にスポーン（無敵時間なし）

### ブロックシステム
- 種類:
  - **通常ブロック** (HP1)
  - **堅いブロック** (HP2〜3)
  - **★ブロック** — 通常/堅いブロックに★マークが描画され、破壊すると★アイテムをドロップ
- 初期配置: 2〜3段
- 画面上部から段が追加され、全体が下にずれる

### 操作体系

#### スマホ操作（メイン）
- 画面左右に親指で操作しやすい位置にボタンを配置
- タップ → 長押しで左右移動
- 左右ボタン同時押しでチャージ（チャージ中はパドル移動不可）
- **同方向ボタン2連打（300ms以内）でダッシュ単発発動**
- 基本技は**スキル名のギザギザ吹き出し**をタップして発動

#### PC操作
- 左右キーで移動
- **同じ方向キーを2連打（300ms以内）でダッシュ単発発動**
- 上キー or **スキル名吹き出しのクリック**で基本技発動
- 左右キー同時押しでチャージ（片方を離せば即移動可能）

### エネルギーシステム（基本技用）

#### エネルギーゲージ
- 0〜100の値。100で満タン
- 左右同時押し（チャージ操作）中に蓄積される
- チャージ中はパドル移動不可 → チャージするか、ボールを追うかの駆け引きが生まれる

#### エネルギータンク
- ゲージが100に達するとタンクが1つ増え、ゲージは0にリセットされる
- 最大保持数: 5（将来的にプレイヤーレベルで増加可能）

#### ダッシュ（旧スピードアップ）
- 同方向キー/ボタン2連打（300ms以内）でパドルが一瞬高速移動する単発アクション
- 1回のダッシュにつきゲージを **25（最大の1/4）** 消費
- ゲージが不足する場合は **タンクを1個消費してゲージを満タンにしてから** 25を引いてダッシュ
- ゲージ・タンクともに不足の場合はダッシュ不発（無反応）
- ダッシュ中は約0.1秒（約160px）瞬間移動。**0.2秒のフェードアウトで残像が複数残り、残像もボールを反射する**（行き過ぎても残像でカバー）
- ピンチを瞬発的にしのげるが、タンクを温存するか発動するかのトレードオフ

#### 基本技（タンク消費）
- プレイヤーが任意のタイミングで発動ボタンを押す
- 発動時のタンク数に応じて異なる基本技が発動（消費タンク＝対応する数）
- 初期実装は3種類:
  - 1タンク: パドル拡大（一定時間）
  - 2タンク: ボール分裂（複数同時）
  - 3タンク: バリア（一定時間、ボール落下防止）
- 4〜5タンクの基本技は将来実装（元案にある「無敵」「最強枠」はここに配置検討）

### ★スキルポイントと必殺技

#### ★のドロップ
- ★を持つブロックを破壊すると、破壊位置から★が落下する
- パドルでキャッチすると★を1つ獲得
- 画面下に落ちると★は消失（取り逃しリスクあり）

#### 必殺技の発動
- ★を3つ集めると、選択キャラクターの固有必殺技が自動発動する
- 発動後は★カウントが0にリセット

#### キャラクター選択
- ゲーム開始時にキャラクター（パドル）を選択する
- キャラごとに固有の必殺技を持つ
- 初期実装は1キャラクターから:
  - キャラクター1: **貫通弾** — パドル真上にエネルギー弾を3秒チャージ → 自動発射。ブロックをHP無視で貫通破壊し、ボスに直撃すれば大ダメージ
